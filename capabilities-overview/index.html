Driver Advisory System

Driver Advisory System is an innovative tool designed to assist train drivers in the delivery of efficient and punctual train services. By integrating a wide range of data sources, including live positioning, infrastructure characteristics, and real-time transport plans, the system calculates and presents the ideal driving profile for each journey.

SBB RASTA Stack

The RASTA Stack is a reference architecture and set of components for railway traffic management and operations systems. It enables the integration of heterogeneous systems, supports real-time data exchange, and provides a foundation for building scalable, interoperable railway applications.

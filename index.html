Capella Collaboration Manager

A web platform for collaboration on MBSE and Capella projects.

DAC Migration DSS

Within the next years around 500 K freight wagons all over Europe owned and operated by various corporations will be converted from screw couplers to digital automatic couplers (DAC). To facilitate the migration process, a decision support system (DSS) is planned under the project.

Driver Advisory System

Driver Advisory System is an innovative tool designed to assist train drivers in the delivery of efficient and punctual train services. By integrating a wide range of data sources, including live positioning, infrastructure characteristics, and real-time transport plans, the system calculates and presents the ideal driving profile for each journey.

libLRS

The goal of the library is to have a flexible, high performance and easy to integrate linear referencing systems (LRS) library that can be used in any system to manipulate LRSs.

MATSim

MATSim (Multi-Agent Transport Simulation) is an open-source framework and set of Java libraries for large-scale agent-based transport simulations, supporting demand modelling, mobility simulation, iterative re-planning, and analysis.

MERITS Transformation Tools

Transformation of both timetable and location data structures between MERITS EDIFACT and CSV in both directions.

MOTIS

multimodal routing, geocoding, and map tiles

Netzgrafik-Editor

Netzgrafik-Editor is now a mature tool for creating and analyzing regular-interval timetables. It’s versatile for logistics planning in various domains. Features include interactive editing, graphic timetables, trainrun editing, and logistics analysis.

nextex-parser-java

Parse NeTEx files and lookup entities in an index

OpenTripPlanner

An open source multi-modal trip planner https://github.com/opentripplanner/OpenTripPlanner

OSDM Demo App

To showcase the power of OSDM the Demo App aims to provide a working app to interesting parties to start implementing their apps and clients. The app covers the complete booking flow, including ancillary services, and supports trains, buses or other means of public transport. The code found here is the second implementation based on the results of the Drei-Länder-Hack 2024. Further development includes the implementation of seat reservations and after-sales

OSRD

OSRD is an open source web application for railway infrastructure design, capacity analysis, timetabling and simulation.

RailLabel

A devkit for working with recorded and annotated train ride data from Deutsche Bahn.

RCM OSS

RCM by SBB comprises a suite of products for rail condition monitoring. Based on the open-source data format RCM-DX (RCM Data eXchange), RCM enables easy accessibility and visualization of railway diagnostic data.

SBB ML Models

Trained YOLOv5 object‑detection models and tooling for running models on mobile (CoreML for iOS, TensorFlow Lite for Android), targeting objects in trains and stations.

SBB RASTA Stack

The RASTA Stack is a reference architecture and set of components for railway traffic management and operations systems. It enables the integration of heterogeneous systems, supports real-time data exchange, and provides a foundation for building scalable, interoperable railway applications.
